Supported Export Formats
========================

This project supports the following export formats.

EPUB
----

Standard EPUB ebook format.

Command: ``save-novel-to-epub``

TXT
---

Plain text format.

Command: ``save-novel-to-txt``

HTML
----

A basic website output with simple reading controls such as font size
adjustments and a chapter index.

Command: ``save-novel-to-html``
